### Setup
```shell
git clone https://github.com/stabilitydao/app.git
cd app
yarn
```

### Development at localhost:3000
```shell
yarn dev
```

### Build
```shell
yarn build
```

### Production
```shell
yarn start
```

### Lint
```shell
yarn lint
```
